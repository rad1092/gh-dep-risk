# Security Policy

If you believe you have found a security vulnerability in `gh-dep-risk`, please
avoid filing a public issue first.

Preferred disclosure path:

- use GitHub's private vulnerability reporting for this repository if it is
  enabled
- otherwise contact the maintainer privately before public disclosure

Please include:

- affected version or commit
- reproduction steps
- impact summary
- any suggested mitigation

The goal is coordinated disclosure and a fix before public discussion where
possible.
